Sistema de Gestión de Préstamos y Laboratorios - ITSSNP

Descripción del Proyecto

Plataforma institucional desarrollada para la Unidad de Prácticas del ITSSNP, diseñada para administrar el préstamo de herramientas a estudiantes y gestionar el acceso a laboratorios de manera eficiente y segura.

🔧 Módulo de Préstamos de Herramientas

  • 📋 Registro y control de préstamos: Préstamo de una o varias herramientas a estudiantes.
  • 🔍 Búsqueda avanzada: Filtrado por nombre, código o categoría de herramienta.
  • 🧑‍🎓 Gestión de estudiantes: Búsqueda por nombre o número de control.
  • 📊 Historial y evaluación: Registro de devoluciones y calificación de estado de herramientas.
  • ⚙️ Manejo de stock: Alta, edición y control de herramientas disponibles.

🏫 Módulo de Control de Laboratorios

  • 🚪 Administración de accesos: Registro de entrada y salida de estudiantes y profesores.
  • 📌 Control detallado: Número de alumnos por laboratorio, género, ocupación del aula.
  • 📚 Historial de uso: Registro de sesiones, profesores asignados y materias impartidas.

🛠 Tecnologías utilizadas:

C#, ASP.NET MVC, SQL Server, JavaScript, Bootstrap 5, AJAX

Chatbot Inteligente con WhatsApp y ChatGPT

🚀 Ventajas y Usos

Este chatbot permite la automatización de consultas en bases de datos mediante mensajes de WhatsApp, facilitando el acceso a información sin necesidad de conocimientos técnicos en SQL.

✨ Beneficios:

  • 💬 Interacción en tiempo real: Responde consultas en WhatsApp de forma instantánea.
  • 🤖 Automatización inteligente: Conversión de preguntas en lenguaje natural a SQL.
  • 📊 Optimización de consultas: Acceso rápido a información sin intervención manual.
  • 🔗 Integración sencilla: Compatible con sistemas empresariales existentes.
  • ⏳ Ahorro de tiempo: Reduce la carga de trabajo del equipo de soporte.

🛠️ Descripción Técnica del Proyecto

API en .NET Core 8.0 que conecta una base de datos con la API de WhatsApp Cloud, permitiendo a los usuarios interactuar con un chatbot que interpreta consultas en lenguaje natural y las convierte en SQL mediante OpenAI.

📌 Características principales:

  • 📡 Conexión con WhatsApp Cloud API: Envía y recibe mensajes automáticamente.
  • 🔄 Webhooks en tiempo real: Procesamiento de mensajes entrantes de forma dinámica.
  • 📝 Conversión de lenguaje natural a SQL: Utiliza ChatGPT para interpretar y ejecutar consultas.
  • 💾 Integración con SQL Server: Consulta bases de datos y devuelve respuestas automatizadas.
  • 🛠️ Pruebas con Postman: Validación y depuración de la API antes del despliegue.

🖥️ Tecnologías utilizadas:

C#, .NET Core 8.0, SQL Server, WhatsApp Cloud API, Webhooks, OpenAI API, Postman

API de Procesamiento de Documentos PDF con Inteligencia Artificial

🚀 Ventajas y Usos

Esta API permite la extracción y análisis de información desde documentos PDF, facilitando la automatización de procesos de lectura y búsqueda de datos clave.

✨ Beneficios:

  • 📄 Extracción automatizada de texto: Convierte archivos PDF en datos estructurados.
  • 🔍 Búsqueda inteligente: Encuentra información relevante dentro de documentos extensos.
  • 🤖 Procesamiento con IA: Usa OpenAI para interpretar y responder consultas sobre el contenido.
  • 📊 Integración sencilla: Compatible con bases de datos y otros sistemas empresariales.
  • ⏳ Ahorro de tiempo: Reduce la necesidad de revisión manual de documentos.

🛠️ Descripción Técnica del Proyecto

API desarrollada en .NET Core 8.0 que permite la carga, análisis y consulta de documentos PDF. Utiliza OpenAI para la interpretación del contenido y ofrece una interfaz para consultas mediante lenguaje natural.

📌 Características principales:

  • 🔄 Analisis de documentos PDF
  • 📝 Consultas en lenguaje natural: Permite realizar preguntas sobre el contenido de los PDFs.
  • 💾 Integración con SQL Server: Guarda y recupera datos extraídos de los documentos.
  • 🛠️ Pruebas con Postman: Validación y depuración de la API antes del despliegue.

🖥️ Tecnologías utilizadas:

Python, Flask, OpenAI API (GPT), Google Cloud Run, Docker, Hugging Face, Web API, Postman, FAISS
